## ✨ 功能特性

- **多标签页管理** — 同时管理多个 WebSocket 连接，互不干扰
- **自动重连** — 连接断开后自动重连，可配置重连次数和间隔
- **心跳检测** — 支持自定义心跳消息内容和间隔，自动检测超时断开
- **消息格式化** — 自动识别并格式化 JSON / XML 消息
- **Hex/Binary 模式** — 支持十六进制数据收发
- **消息历史** — 消息记录自动保存（localStorage），支持搜索、过滤、导出
- **请求-响应配对** — 自动将发送消息与响应分组显示
- **快捷键** — `Ctrl+Enter` 发送、`Ctrl+T` 新建标签、`Ctrl+W` 关闭标签
- **配置导入/导出** — 一键备份和恢复所有连接配置和历史
- **暗色主题** — 护眼暗色 UI，适配桌面和移动端
